# 临时任务管理

临时方案执行 + 临时脚本管理，执行完毕后自动清理

## 使用方式

### 方案执行
```
/plan-exec <任务描述>
```

### 脚本管理
```
/plan-exec run <脚本名>
```

## 执行流程

### 方案执行
1. 理解需求，制定方案
2. 保存方案到 `.claude/tmp/plan-{timestamp}.md`
3. 逐步执行方案
4. 执行完毕，删除临时文件
5. 汇报结果

### 脚本管理
1. 识别临时脚本（`_` 开头的 Python 文件）
2. 运行脚本
3. 记录输出到 `private/logs/`
4. 删除脚本
5. 汇报结果

## 示例

```
/plan-exec 添加用户登录功能
/plan-exec run _check.py
```
